# Sweet Haven Café

A beautiful, standalone boutique café website featuring artisanal pastries and premium coffee.

## Features

- **Gourmet Menu**: Explore our categories of pastries, cakes, desserts, and beverages.
- **Visual Gallery**: A curated collection of our cozy atmosphere and delicious creations.
- **Responsive Design**: Optimized for both mobile and desktop experiences.
- **Reservation System**: Easy access to contact information and table booking.

## Tech Stack

- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Lucide React** (Icons)
- **Framer Motion** (Animations)

## Run Locally

**Prerequisites:** Node.js (v18+)

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the website:**
   Visit `http://localhost:3000` in your browser.
